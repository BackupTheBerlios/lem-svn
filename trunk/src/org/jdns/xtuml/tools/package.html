LEM Parser

The LEM parser is a central plank in the xtUML toolkit. The parser comprises:

  1. xtUML meta-model
  2. LEM API - a JAVA-based relisation of the xtUML meta-model
  3. Language definition (LEM grammar)
  4. LEM parser (based on JavaCC)
  5. 3 pass model builder - creates an in-memory object graph which is an meta-model instatiation faithfully representing the LEM-coded input model

The LEM API provides access to the in-memory representation of the executable model. This API provides a foundation for a rich set of xtUML tools based on LEM. These include static and dynamic verifier, code translators, compilers and documentation generators.
